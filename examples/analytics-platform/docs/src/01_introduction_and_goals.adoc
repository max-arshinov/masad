== Introduction and Goals

=== Requirements Overview

- Analytics platform is a system that tracks events from websites: page view, button click, etc.
- Track the total number of visitors, the number of unique visitors, broken down by: hours, days, weeks, months, geolocation, traffic source, browser, device, and UTM tags.

=== Quality Goals

- Reports and charts must be delivered within 1.5 seconds for any combination of parameters over a period not exceeding 3 months.
- The system must be able to serve approximately 100 billion events per day.
- Delays of 50–100 minutes are acceptable between a user action and its appearance in statistics.
- Data per website must be stored “forever.”

See <<section-quality-scenarios,Quality Requirements>> for the Quality Tree and Scenarios.
