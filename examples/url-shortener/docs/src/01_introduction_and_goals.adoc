=== Requirements Overview

You are tasked to create a URL shortener service similar to bit.ly, with the following quality requirements:

=== Quality Goals

- 1 million new users per year, each user shortens an average of 100 links per year.
- Links must be stored for 3 years. Redirecting to the target address must take no more than 0.2 seconds.
- Click statistics must be tracked.
